Вот максимально упрощённая версия страницы с тремя цветными квадратами, без лишних эффектов и сложного дизайна: ```html Цветные квадраты
``` ## Что делает программа: 1. **Три квадрата** — красный, зелёный, синий 2. **При клике** на любой квадрат фон страницы становится такого же цвета 3. Квадраты имеют рамку для наглядности ## Как это работает: - `querySelectorAll` — находим все элементы с классом `square` - `addEventListener('click', ...)` — ждём клика на каждом квадрате - `getComputedStyle` — получаем настоящий цвет квадрата - `document.body.style.backgroundColor` — меняем цвет фона страницы Готово — просто, понятно, работает!
